## üöÄ How to Use

### Setup (First Time)

1. **Get Your API Key**
   - Visit [Google AI Studio](https://aistudio.google.com/app/apikey)
   - Click "Create API Key" (it's free!)
   - Copy your API key

2. **Enable the API**
   - Go to [Google Cloud Console](https://console.cloud.google.com/apis/library/generativelanguage.googleapis.com)
   - Click "ENABLE" for the Generative Language API
   - Wait 1-2 minutes for activation

3. **Configure Tod AI**
   - Click the floating AI assistant button
   - Click the Settings icon (gear) in the chat header
   - Paste your API key
   - Click "Save"
   - The system will automatically test the connection

### Using the Assistant

1. **Open the Chat**
   - Click the sparkle button in the bottom-right corner
   - The chat panel slides in from the right

2. **Ask Questions**
   - Type your question in the input box
   - OR click the microphone to speak
   - Press Enter or click Send

3. **Get AI-Powered Help**
   - Ask anything: homework help, concept explanations, study tips
   - Request summaries, quiz questions, or learning strategies
   - Get navigation help or feature explanations

### Example Questions

- "Can you help me understand fractions?"

## üÜò Troubleshooting

### "API Key Error"
- Verify your API key is correct
- Ensure Generative Language API is enabled
- Wait 1-2 minutes after enabling the API
- Try getting a new API key

### "No Response"
- Check your internet connection
- Verify API key is saved in settings
- Check if you've hit rate limits
- Try again in a few moments

### "Voice Not Working"
- Grant microphone permission in browser
- Check browser compatibility (Chrome/Edge work best)
- Ensure microphone is connected and working

### Chat Not Opening
- Refresh the page
- Clear browser cache
- Check browser console for errors
